## Tentang Simple Ticket
Simple Ticket adalah aplikasi berbasis web sederhana untuk manajemen ticketing pribadi. Aplikasi ini memungkinkan pengguna untuk membuat, memantau, dan menyelesaikan ticket dengan mudah, membantu meningkatkan produktivitas dan organisasi tugas harian.
